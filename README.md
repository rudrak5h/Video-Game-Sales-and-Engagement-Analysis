# 🎮 Video Game Sales & Engagement Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing video game sales, engagement metrics, ratings, and market trends using Python, SQL, Streamlit, and Power BI.

The project combines:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* SQL Analysis
* Interactive Dashboard Development
* Business Insights Visualization

The goal is to transform raw gaming datasets into meaningful insights through analytics and interactive dashboards.

---

# 📂 Dataset Information

The project uses multiple datasets:

### 1. games.csv

Contains:

* Game titles
* Ratings
* Reviews
* Wishlist counts
* Plays
* Backlogs
* Engagement metrics

### 2. vgsales.csv

Contains:

* Global sales
* Regional sales
* Platforms
* Publishers
* Genres
* Release years

### 3. merged_games_sales.csv

Merged dataset used for:

* Analysis
* Visualization
* Dashboard creation
